# Finding Sports Facilities 

Making an app to map local sports facilities sounds simple enough. We show and target public access for common grounds like football pitches, gyms, or tennis courts near your home. Actually, finding a place to train shouldn't be difficult gathering feeds from Bristol Open Data ensures the records stay up to date and accurate for everyone involved. Relying on these verified streams forces our software to stay lean while serving relevant results. Users can finally pinpoint exactly where they need to go without scrolling through endless static lists.
